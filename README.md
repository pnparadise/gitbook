## 简介

轻快，简洁，易用， Node + Vuejs + CodeMirror 开发的Github Flavored Markdown Editor + 微博客预览。

基于github api 数据接口，markdown文档直接同步到repository 实时预览+实时更新 打造 `serverless` 轻应用 

考虑到数据安全 github授权api拒绝cros请求 授权服务器搭建必不可少
